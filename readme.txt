== Description ==

<strong>Social Media Sharing by FVP</strong> is a <strong>Free WordPress plugin</strong> to allow anyone easily share website content in most popular social media (Facebook, Twitter, Google+, Pinterest, LinkedIn, Whatsapp and Email) on your website.

A simple but powerful plugin to make your website social-share friendly, without impact in the speed of your site.

You can select the social media profiles you want to activate and select one of the designs from 13 beautiful available styles.

The available social media are: Facebook, Twitter, Google+, Pinterest, LinkedIn. Also there is the option to share by Whatsapp and Email.

Just install, customize and that's it!

== Installation ==

1. Upload the plugin files to the /wp-content/plugins/fvp-social-media directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Social Media FVP->Settings screen to configure the plugin
4. Use the Social Media FVP->Customize screen to customize your sharing buttons


== Frequently Asked Questions ==

= What does this plugin do? =
This plugin provides the ability to allow anyone easily share your website content in most popular social media.

= Will it effect my site's speed? =
No , because we use a short css code, and we do not use javascript to display buttons.
